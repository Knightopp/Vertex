# Vertex
Vertex is a game library tracker and partially a launcher made using antigravity.Main purpose was to track the games/even cracked unofficial games.It shows stats about how much you played it, you are able to put a review on the game.

# Compatability 
Vertex is currently supported by Windows and Android.
While the game tracker is only for windows, android version is just to view or edit your library.

# Features
- **Unified Library:** Automatically detects installed steam games when they run.
- **Cross Platform:** Your library will be cloud synced to your mobile phone.
- **Auto-Updating:** Built-in auto-updater ensures you always have the latest features without manual installation.
- **Rich Metadata:** Pulls the covers/descriptions and metadata directly from SteamGridDB and IGDB.
- **Deep-Linking:** Log into your account using google or discord to sync your library to cloud using secure OAuth deep linnk integration

- **[Download for Windows (.exe) or Android (.apk) here!](https://github.com/Knightopp/Vertex/releases/latest)**

- ### Auto-Updates
- **Windows:** Vertex will automatically check for updates in the background. When a new update is available, you will receive an in-app notification. Simply click "Install" and it will seamlessly update and restart in seconds!
- **Android:** Vertex will check for updates on startup. If a new version is available, it will prompt you to download the latest `.apk` directly to your phone.

- ##  Built With
Vertex is built using a modern, blazing-fast web stack powered by Tauri v2:
- **Tauri v2:** The lightning-fast desktop/mobile runtime built in Rust.
- **React (Vite):** Fast frontend framework.
- **TailwindCSS & Radix UI:** For gorgeous, accessible, and responsive styling.
- **Supabase:** For secure cloud data syncing and Google OAuth authentication.
- **SQLite:** For local, offline-first database storage.
